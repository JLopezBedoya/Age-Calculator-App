## Instalación

1. Clona este repositorio:

```bash
git clone https://github.com/Jlopezbedoya/Age-calculator-app.git
cd Age-calculator-app
```

2. Instalar las dependencias

```bash
npm install
```

3. Inicia la aplicación:

```bash
npm run dev
```
